Answer 1

The primary difference between an argument and a parameter is that a parameter is a variable or set of variables in a method. The argument is simply the actual input value of the parameter. You would use a parameter when the data is unknown yet or the argument must be entered by the user. Arguments can be used if there is no variation and the values are already known.
